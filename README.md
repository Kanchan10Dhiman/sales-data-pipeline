# End-to-End Data Engineering Pipeline

## 📌 Project Overview
This project demonstrates a simple data pipeline using Python and SQL concepts. The goal is to process raw sales data and generate meaningful business insights.

---

## ⚙️ Workflow (Pipeline Stages)

### 1. Data Ingestion
- Loaded raw data from CSV file (`sales.csv`)

### 2. Data Processing (Python)
- Removed duplicates
- Handled missing values
- Performed aggregations using Pandas

### 3. Data Analysis (SQL Concepts)
- Used GROUP BY for aggregation
- Identified top-performing products and cities

---

## 📊 Key Insights
- Top Selling Product: Laptop  
- Highest Sales City: Delhi  
- Total Sales calculated from dataset

- Python
- Pandas
- SQL (conceptual understanding)
